# Excel_BankLoan_Analysis
 CreditPulse – Bank Loan Performance & Risk Analysis Dashboard
📌 What It Is:

An advanced Excel-based analytics dashboard built on 38,576 real bank loan records worth $435.7M in total funded amount
Consists of a dual-dashboard system — a Summary Dashboard for KPI tracking and an Overview Dashboard for deep-dive trend analysis
Covers 25 data variables including loan grade, borrower income, DTI ratio, employment length, interest rate, and repayment status
Classifies every loan into Good Loan vs Bad Loan categories to enable portfolio-level risk visibility


🔧 What I Did:

Cleaned and transformed raw, unstructured financial data across 38,000+ records into a structured analytical model
Built dynamic Pivot Tables, KPI cards, donut charts, bar charts, and slicers for interactive filtering
Engineered key metrics including Total Funded Amount, Amount Received, Avg Interest Rate (12%), and Avg DTI (13.3%)
Segmented loan data by purpose, grade, home ownership, term, and state for multi-dimensional analysis


❗ What Problem It Solved:

Banks struggle to identify which loans are likely to default before they become financial losses
Manual loan reporting is time-consuming and lacks the visual clarity needed for quick executive decisions
There was no single unified view connecting borrower profiles, loan health, and repayment behavior in one place
This dashboard solved all three by creating a centralized, real-time-ready risk monitoring system


📈 Impact & Outcome:

Revealed that 13.8% of loans (Bad Loans) account for $65.5M in high-risk funded amount — a critical finding for risk teams
Enabled faster credit decisions by surfacing patterns across loan grades A–G and 36 vs 60-month terms
Identified Debt Consolidation as the #1 loan purpose (47%) — helping prioritize product and risk strategy
Delivered an executive-ready dashboard that replaces hours of manual reporting with instant, filterable insights
